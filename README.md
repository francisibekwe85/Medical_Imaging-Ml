# Medical Imaging ML 🏥

A comprehensive collection of machine learning projects focused on medical image analysis and healthcare diagnostics using Jupyter Notebooks.

## 📋 Overview

This repository contains multiple machine learning projects that leverage computer vision and deep learning techniques to analyze various medical imaging datasets. Each project is implemented as a Jupyter Notebook with end-to-end workflows including data preprocessing, model training, evaluation, and visualization.

## 🎯 Projects

### 1. **Brain Tumor Classification** 
   - **File:** `BRAINTUMOR_PROJECT.ipynb`
   - **Purpose:** Detection and classification of brain tumors from MRI scans
   - **Size:** ~366 KB

### 2. **Breast Cancer Detection**
   - **File:** `BREAST_CANCER_PROJECT.ipynb`
   - **Purpose:** Classification and analysis of breast cancer from mammography images
   - **Size:** ~2.6 MB

### 3. **Chest X-Ray Analysis**
   - **File:** `CHESTRAY_PROJECT2.ipynb`
   - **Purpose:** Diagnosis and classification of chest conditions from X-ray images
   - **Size:** ~3.5 MB

### 4. **Fetal Health Assessment**
   - **File:** `Fetal_Health_Project.ipynb`
   - **Purpose:** Evaluation of fetal health through ultrasound imaging analysis
   - **Size:** ~538 KB

### 5. **Kidney Stone Detection**
   - **File:** `Kidneystone_model_Evaluation.ipynb`
   - **Purpose:** Model evaluation and assessment for kidney stone detection
   - **Size:** ~32 KB

### 6. **Leukemia Classification**
   - **File:** `Leukemia_project.ipynb`
   - **Purpose:** Blood cell image analysis for leukemia classification
   - **Size:** ~268 KB

### 7. **Pregnancy Health Monitoring**
   - **File:** `Pregnancy_Project.ipynb`
   - **Purpose:** Analysis of pregnancy-related health indicators
   - **Size:** ~88 KB

### 8. **Retinopathy Detection**
   - **File:** `Rethinopathy_Project.ipynb`
   - **Purpose:** Detection and grading of diabetic retinopathy from retinal images
   - **Size:** ~1.7 MB

## 🚀 Getting Started

### Prerequisites
- Python 3.7+
- Jupyter Notebook
- Required libraries (see Technologies section)

### Installation

1. Clone the repository:
```bash
git clone https://github.com/francisibekwe85/Medical_Imaging-Ml.git
cd Medical_Imaging-Ml
```

2. Install required dependencies:
```bash
pip install jupyter numpy pandas scikit-learn opencv-python tensorflow matplotlib seaborn
```

3. Launch Jupyter Notebook:
```bash
jupyter notebook
```

4. Open any of the project notebooks and run the cells sequentially.
